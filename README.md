MoodTrack v1.0

MoodTrack nace de personas con la necesidad de registrar sus estados emocionales en el momento, con el fin de analizar sus cambios de humor y el manejo del temperamento.

Una herramienta útil para que psicólogos se la recomienden a pacientes con TDAH, ansiedad o bipolaridad; o simplemente personas con problemas de la ira.

MoodTrack evitara los problemas de dejar todo escrito en libretas o notas en el celular, susceptibles a pérdidas de esos datos, o no poder llevar un registro organizado adecuadamente.

Objetivos:
    - Diseño limpio y tranquilo para los usuarios. ✓
    - Gestión de usuarios (Iniciar sesión, registrar usuario). ~
    - Gestión de registro para emociones (Crear nuevo estado emocional, buscar estados emocionales). ~
